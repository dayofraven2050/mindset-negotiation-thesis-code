# mindset-negotiation-thesis-code
本科毕业论文《思维模式对谈判策略选择的影响》相关代码
## 代码说明

本仓库包含本科毕业论文《思维模式对谈判策略选择的影响》中的开放题文本分析与稳健性检验代码。

主要功能包括：

1. 读取问卷数据并自动识别相关变量；
2. 对开放题文本进行清洗和有效性筛选；
3. 基于规则词典将开放题回答划分为六类出价理由；
4. 构造 rule_E（强硬压价/优势最大化）等文本分类变量；
5. 检验 rule_E 与攻击性谈判策略变量之间的关系；
6. 进行回归分析、卡方检验、Mann-Whitney U 检验、Kruskal-Wallis 检验等稳健性分析；
7. 使用 TF-IDF 和 BERT 文本向量进行分类与预测分析；
8. 将主要分析结果导出为 Excel 表格。

由于原始问卷数据涉及被试隐私，本文不公开上传原始数据。
